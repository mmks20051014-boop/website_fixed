01 About

CodeRoute 90について

小林裕斗

最短距離で「未経験」を脱却する攻略ログ

CodeRoute 90が扱うのは「最短で結果を出すための現実」

CodeRoute 90は、40歳・文系営業職から90日でエンジニア転職した運営者が、その過程で得た「最短攻略ルート」を共有するメディアです。
独学推奨の空気や"夢のある美談"はできるだけ排除し、効率最優先で前に進むための判断材料を整理して公開します。


未経験からの転職と、キャリアアップの「最適ルート」

挫折や停滞の原因は、難易度よりも「時間の浪費」や「間違った努力」にあります。
未経験から90日でエンジニア転職した自身の経験を踏まえ、
転職活動からキャリアアップの戦略まで、投資対効果が出る方法を事実ベースで整理します。


90日間の「技術習得、転職ロードマップ」

Phase 1（1〜30日）：HTML/CSS/JSで静的サイト制作 → Webの構造を身体で理解と同時に転職活動スタート
Phase 2（31〜60日）：Python/Djangoでアプリ開発 → DB/バックエンドの勘所を掴む
Phase 3（61〜90日）：設計〜デプロイ〜SEO → 公開して動かすところまで完走、転職成功

成功体験だけでなく、つまずいた箇所も含めて、「どこで詰まり、どう抜けたか」を再現できる形で記録します。

このサイトで目指せる到達点

知識ゼロからでも、学習→制作→公開までを線でつなぎ、「未経験」を脱却するための道筋が見える

扱う領域：Web制作 / Webアプリ開発 / Python / JavaScript / デプロイ / SEO
